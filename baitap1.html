What is CSS?

Cascading Style Sheets (CSS) us used to format the layout of a webpage.

With CSS, you can control the color, font, the size of text, the spacing between elements, how elements are positioned and laid out, what backgroudnd images or background colors are to be used, different displays for different divices and screen sizes, and much more!

Tip: The world cascading means that a style applited to a parent element will aslo apply to all chirlden elements within the parent. So, if you set the color of the body text to "blue", all headings, paragraphs, and orther text elements within the body will aslo ghe the same color (unless you specify something else)!
